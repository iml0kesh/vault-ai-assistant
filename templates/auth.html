Auth Methods

AppRole Auth AWS IAM Auth Kubernetes Auth GitHub Auth

Vault supports multiple authentication methods. Choose the one that best fits your environment:

- AppRole: Ideal for machine-to-machine authentication.
- AWS IAM: Best for AWS EC2, Lambda, or ECS workloads.
- Kubernetes: Automatically authenticate pods using service account tokens.
- GitHub: Used mainly for developer access or CI/CD pipelines.
